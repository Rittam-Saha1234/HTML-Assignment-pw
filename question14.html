Ordered List

  1. <html>: This tag is used to define the root of an HTML document.
  2. <head>: This tag contains meta-information about the HTML document, such as title, scripts, styles, and more.
  3. <title>: This tag specifies the title of the HTML document, which is displayed in the browser's title bar or tab.
  4. <body>: This tag contains the content of the HTML document, including text, images, links, and other elements.
  5. <h1> to <h6>: These tags are used to define headings of different levels, with <h1> being the most important and <h6> being the least important.
  6. <p>: This tag is used to define a paragraph of text.
  7. <a>: This tag is used to create hyperlinks to other web pages or resources.
  8. <img>: This tag is used to embed images into the HTML document.
  9. <ul>: This tag defines an unordered list, typically used for bullet-point lists.
  10. <ol>: This tag defines an ordered list, typically used for numbered lists.
